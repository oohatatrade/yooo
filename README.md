# 秋田県 契約結果取得アプリ

秋田県電子入札システムの「契約結果情報」と「発注情報」から検索結果を取得し、Excelファイルにまとめるローカルアプリです。

## 起動

```powershell
& 'C:\Users\MRU-OHATA\.cache\codex-runtimes\codex-primary-runtime\dependencies\python\python.exe' app.py
```

起動後、ブラウザで `http://127.0.0.1:8765` を開きます。

## できること

- 契約年度、調達区分、工事・委託名称キーワード、場所、請負者名で検索
- 取得種別を「契約結果情報」と「発注情報」から選択
- 複数キーワードを改行、カンマ、読点区切りで一括取得
- ページングされた結果を全ページ取得
- 契約結果一覧と検索条件を `.xlsx` に出力

Excelファイルは `output` フォルダに作成されます。
